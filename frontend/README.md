# 🎨 Disease Predictor – Frontend

This is the **frontend application** of the Disease Predictor project, built using **React + Vite**.  
It provides a simple and interactive UI where users can enter input data and get predictions from the backend ML model.  
The app also integrates with **Google Stitch** for embedding and AI workflow connectivity.

---

## ✨ Features

- ⚡ **Fast and lightweight React + Vite setup**  
- 🖥️ **User-friendly interface** for disease prediction  
- 🎨 Modern styling with **CSS / PostCSS**  
- 🔗 **Google Stitch integration** for embedding & AI pipeline  
- 🔥 Hot Reloading with Vite for faster dev experience
